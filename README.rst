=========
WS Celery
=========

Real time celery monitoring using websockets. Inspired by `flower <https://github.com/mher/flower>`__.

************
Requirements 
************

* Python 3.x

**********
Quickstart
**********

::

    $ python setup.py install
    $ wscelery

**********
Caveats
**********

* TLS encryption not (yet) supported
